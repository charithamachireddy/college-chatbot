# 🎓 Sanskrithi School of Engineering Chatbot

An AI-based chatbot designed to provide information about **Sanskrithi School of Engineering College** such as courses, fees, faculty, and general queries.



##  Features
-  Chatbot interface for user queries  
-  College-related information (courses, fees, faculty)  
-  Blue-themed UI with background image  
-  Fast and simple responses  
-  Built using Flask (Python backend)


##  Tech Stack
- Frontend: HTML, CSS  
- Backend: Python (Flask)  
- Version Control: Git & GitHub 

##  Future Improvements
-  AI-based intelligent chatbot  
-  User login system  
-  Mobile responsive design  
-  Voice chatbot integration
